## Workflow

### Step-by-Step Process

1. **Data Ingestion**
   - Place raw data files in `1. Raw Dataset/`
   - Validate file format and structure

2. **Data Cleaning**
   - Execute cleaning scripts
   - Save validated data to `2. Cleaned Dataset/`

3. **Analysis & Calculations**
   - Perform statistical computations
   - Generate pivot tables in `3. Calculation & Pivot table/`

4. **Visualization**
   - Build interactive dashboards
   - Store visualizations in `4. Dashboard/`

5. **Documentation**
   - Document methodology and findings
   - Archive in `5. Documentation/`

6. **Presentation**
   - Prepare executive summaries
   - Finalize deliverables in `6. Presentation/`
